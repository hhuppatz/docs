## Quick start

<Steps>
  <Step title="Open the Visual Page Builder">
    Navigate to the page you want to edit in Squiz Matrix and open it in the Visual Page Builder.
  </Step>
  <Step title="Add a component">
    Click the **Add component** button (or add a content block and then click **Add component**) to open the component browser.
  </Step>
  <Step title="Choose a component">
    Search for any component prefixed with **HH -** (e.g. *HH - Accordion*, *HH - Banner*) and click **Select**.
  </Step>
  <Step title="Fill in the fields">
    Use the component's editable fields to add your content. Refer to the individual component guides for field-by-field instructions.
  </Step>
</Steps>

<Tip>
  Each component guide includes a section showing common configurations and variant examples so you can see what's possible before you start editing.
</Tip>
