The UltraCX Component Library gives you four ready-to-use, custom-built components for Squiz Matrix. Whether you're building a structured FAQ section, a striking page header, a featured quote, or a linked content card, each component is designed to drop straight into your pages through the Visual Page Builder — no coding required.

## Who this library is for

These components are built for **content editors** working inside the Squiz Matrix Visual Page Builder. If you manage pages through the Matrix interface and use the page builder to add and arrange content blocks, this library is your toolkit. You don't need to touch any code to use any of the four components.

## What's in the library

The library contains four components. Each one is available in the Visual Page Builder's component browser under the **'HH –'** prefix, so you can always find them quickly by typing `HH` in the search field.

## How the components appear in Matrix

All four components are registered under the **`hh-components`** namespace and appear in the Visual Page Builder's component browser with the **'HH –'** prefix:

| Component browser name | Guide page |
|---|---|
| HH – Accordion | [Accordion](/components/accordion) |
| HH – Banner | [Banner](/components/banner) |
| HH – Block Quote | [Blockquote](/components/blockquote) |
| HH – Card | [Card](/components/card) |

When you add one of these components to a page, the builder presents a form with clearly labelled fields — such as headings, body text, images, and links — that you fill in directly. Every field supports inline editing in the Visual Page Builder's preview column, so you can see your changes take effect as you type.

<Tip>
  All UltraCX components support inline editing in the Visual Page Builder. Click any text or media element in the preview pane to edit it in place, without switching to a separate form view.
</Tip>
